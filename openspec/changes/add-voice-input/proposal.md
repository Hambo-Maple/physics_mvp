# Change: 集成百度语音识别实现语音输入

## Why
当前项目仅支持文字输入，用户需要手动打字提问。为了提升交互便利性，特别是在移动端或需要快速输入的场景，需要集成语音识别功能，让用户通过语音快速提问。

## What Changes
- 前端集成 MediaRecorder API 实现录音功能
- 新增语音处理工具（录音启停、音频转 Base64、后端接口调用）
- 后端新增语音识别接口，转发百度智能云语音识别 API
- 语音按钮状态管理（正常/录音中/识别中）
- 识别结果自动填充输入框并触发发送
- 非阻塞式错误提示（文字提示，3秒自动消失）
- 视觉反馈匹配墨绿色主题

## Impact
- 新增能力：`voice-input`（语音输入）
- 修改能力：`chat-interaction`（对话交互）
- 影响文件：
  - `src/utils/voice.js`（新建，语音处理工具）
  - `src/components/ChatBox.vue`（集成语音输入）
  - `src/assets/ChatBox.css`（语音按钮状态样式）
  - `server.js`（新增语音识别接口）
  - `.env`（新增百度 API 密钥配置）
